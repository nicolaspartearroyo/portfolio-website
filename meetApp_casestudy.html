Case Study meetApp

The Challenge

To create a serverless PWA using React based on the features that had been provided by CareerFoundry.

The Process

My process was to take the features provided by CF and write user stories. The idea of the exercise was to implement test-drive development approach which means test first, code later. We also implemented serverless functions, some data visualisation and tools like Jest orEnzyme for unit and integration test and Puppeteer for end-to-end test. Cucumber for user acceptances test and CI/CD for all performance monitoring purposes.

The Goal

A serverless PWA using Create React App that can display and filter meetup events from the Google calendar API. It is developed using Test-Drive-Development techniques to ensure a high test coverage rate.
